# Original Project Requirements

## Milestone 1 Scope
- Build an inventory ingestion pipeline with PySpark.
- Use raw/unstructured inventory source records.
- Clean, validate, transform, and load into MongoDB.
- Produce execution logs and schema validation documentation.

## Future Scope (Milestones 2-4)
- M2: Auth/Product/Cart/Order microservices.
- M3: Semantic search with embeddings and similarity retrieval.
- M4: LLM Ops (versioning, monitoring, structured logging, controlled redeployments).
